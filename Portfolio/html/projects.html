Dance Academy Database (University-Database)

Oracle SQL database designed to manage instructors (choreographers), lessons, and performance schedules, using primary/foreign keys and a junction table to model relationships.

What I did

  • Brainstormed the domain and translated requirements into a relational schema.
  • Designed an ERD and converted it into normalized tables (entities + attributes).
  • Implemented the schema in Oracle SQL using CREATE TABLE, primary keys, and foreign keys.
  • Modeled relationships using a junction table (LS) to map lessons and choreographers.
  • Inserted sample data and verified integrity by running SELECT queries in Oracle APEX.
  • Organized scripts into separate SQL files (schema / data / queries) and published to GitHub.

PetShop Management System (University—Java OOP)

Console-based pet shop management system built in Java to practice OOP principles (inheritance, interfaces, encapsulation) and file I/O for maintaining a simple transaction history.

What I did

  • Designed an OOP model with a base Pet class and derived Cat / Dog classes (inheritance).
  • Implemented interfaces (PetOperations, PetShopOperations) to define required operations and keep the design consistent.
  • Built a PetShop manager to add, remove, search, and display pets (collection management).
  • Created buy/sell style workflows with validation (quantity updates, basic checks).
  • Added file I/O using FileReadWriteDemo to write/read a history log (History.txt).
  • Created a menu-driven entry point in Start.java to run the system from the terminal.
